# Live Grading Assistant - Local Run Instructions

This application uses Service Workers, which require an HTTP server (it won't work correctly if you just open the file directly in the browser).

## How to Run

1.  Open your terminal in this directory.
2.  Run the start script:
    ```bash
    sh start_app.sh
    ```
3.  Open your browser to: [http://localhost:8000/index.html](http://localhost:8000/index.html)

## Requirements
- Python 3 (installed by default on macOS)
- A modern web browser (Chrome, Safari, Firefox)
